# Basic-3D-Editor
Simple 3D Editor I made to get used with C++ \
It took around 3-4 days to make \

# Keybinds

I, O --> Zoom In/Out \
WASD --> Rotates the mesh \ 
Arrows --> Moves the camera \ 
Q, E --> Moves selected vertices in X axis \
R, F --> Moves selected vertices in Y axis \
T, G --> Moves selected vertices in Z axis \
